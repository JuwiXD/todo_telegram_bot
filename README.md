# Telegram To-Do List Bot 📋

Простой, но функциональный Telegram-бот для управления списком дел (To-Do list), написанный на Python с использованием библиотеки `python-telegram-bot`. Бот позволяет добавлять, просматривать и удалять задачи прямо в чате.

## 🚀 Возможности

- 📝 **Добавление задач** — добавьте новую задачу в список одной командой.
- 📋 **Просмотр списка** — выведите все текущие задачи с их порядковыми номерами.
- ❌ **Удаление по номеру** — удалите конкретную задачу, указав её номер.
- 🧹 **Полная очистка** — удалите все задачи сразу одним нажатием.
- 🛡️ **Обработка ошибок** — бот не вылетает, если ввести текст вместо числа или не указать задачу.

## 📂 Стек технологий

- **Язык:** Python 3.10+
- **Библиотека:** [python-telegram-bot](https://python-telegram-bot.org/) (v20+)
- **Хранение данных:** Оперативная память (список Python) — данные сбрасываются при перезапуске бота.

## 📋 Доступные команды

| Команда | Описание | Пример |
| :--- | :--- | :--- |
| `/start` | Показать приветственное сообщение и список команд | `/start` |
| `/add <задача>` | Добавить новую задачу в список | `/add Купить хлеб` |
| `/show` | Показать все текущие задачи с номерами | `/show` |
| `/remove <номер>` | Удалить задачу по её номеру из списка | `/remove 2` |
| `/clear` | Полностью очистить список задач | `/clear` |
